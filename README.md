# Netflix Data Analysis & Recommendation System

## Overview
Exploratory Data Analysis and ML-based recommendation system on Netflix dataset.

## Features
- Data cleaning
- Genre & Director analysis
- Duration insights
- Monthly & yearly trends
- Correlation heatmap
- WordCloud visualization
- Content-based Recommendation System

## Technologies
Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## Machine Learning
TF-IDF + Cosine Similarity

## How to Run
pip install -r requirements.txt
Open notebook and execute cells.

## Insights
- Movies dominate Netflix content
- Drama is most frequent genre
- Most TV shows have only 1 season
- Netflix content peaks after 2018
